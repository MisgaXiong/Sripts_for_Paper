The lineage-level variation identification function was developed to find some feature mutations specific to some lineages of interested of a pathogen like SARS-CoV-2, SARS-CoV-1 but not Dengue etc. Pathogens which can perform this analysis should have small number of mutation among different lineages, rather than large segments of variations.
