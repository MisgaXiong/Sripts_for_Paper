PCR Primer design
Design the regular PCR primers/probe using a target sequence.
Conserve score file (optional)
Primer type
Primer design function can design the regular nucleic acid detection primers/probe for any pathogens. The conserve score file is an optional file that is specially used for the highly conserved primer/probes development for a certain pathogens. This file can be obtained by the Conserved Sequence Determine of PHDtools.
