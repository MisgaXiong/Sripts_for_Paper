Pathogen genomes for typing
The RNA virus such as SARS-CoV, Dengue can use this model to strain typing. In order to get a good result, only the complete or nearly full-length of the genome should be used. The genomes of the pathogen can be merged in one fasta file or in each single fasta file. Please use a .zip or .gz format file to package the fasta file(s).

The DNA pathogen such as bacteria, ASFV and Poxvirus can use this model to strain typing. This approach can not only analyze the complete genome of the isolate, but also can process the draft genome of the isolate. Each genome of the isolate must be in a single fasta file. Please use a .zip or .gz format file to package the multiple fasta files.
The strain typing approach can not only find the most identical strains to your target isolate, but also can cluster differential lineages in the same pathogen. In order to perform the analysis correctly, please ensure that the RNA virus genomes file to be uploaded does not exceed 40Mb and the DNA pathogen genomes file to be uploaded does not exceed 65Mb.
