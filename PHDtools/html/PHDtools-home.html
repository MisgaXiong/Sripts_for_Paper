Pathogen High-dimentional Data tools

PHDtools is an online and user friendly bioinformatics analysis platform which can be used to rapid identify pathogen X in epidemics. By using multiple functions of PHDtools, the accurate, valid and informative pathogen genomics results can be archived. PHDtools will be meaningful to provide guidance for epidemics diagnosis, pathogen genomes research, antiviral drug development and vaccince design! Author: Dongyan Xiong
